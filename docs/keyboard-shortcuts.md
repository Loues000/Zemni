# Keyboard Shortcuts

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + G` | Generate content (summary/flashcards/quiz) |
| `Ctrl/Cmd + B` | Toggle history sidebar |
| `Ctrl/Cmd + K` | Focus search in history sidebar |
| `Ctrl/Cmd + C` | Copy summary (when preview is visible and not editing) |
| `Esc` | Close modals, sidebar, or dialogs |

## Flashcards Mode

| Shortcut | Action |
|----------|--------|
| `Space` / `Enter` / `F` / `↑` / `↓` | Flip flashcard |
| `←` / `→` | Navigate to previous/next flashcard |
| `Esc` | Close flashcard player |

## Quiz Mode

| Shortcut | Action |
|----------|--------|
| `1-9` / `A-Z` | Select answer option |
| `↑` / `K` | Move focus up |
| `↓` / `J` | Move focus down |
| `←` | Previous question |
| `→` / `N` | Next question (or generate more) |
| `R` | Reveal/hide answer |
| `Enter` / `Space` | Select focused option |

## Notes

- Shortcuts are disabled when typing in input fields or text editors
- On Mac, use `Cmd` instead of `Ctrl`
- Some shortcuts may vary based on your keyboard layout
